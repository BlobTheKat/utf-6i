# Utf-6i
encode small!

### Usage
First, we'll import it:
```
npm install utf-6i
```
```js
const UTF6I = require("utf-6i")
```
To encode a string of text, use `<UTF6I>.encode(txt)`:
```js
const UTF6I = require("utf-6i")
let encoded = UTF6I.encode("Hello, World!")
console.log(encoded) //"ﲁ䰰ￜϵ콈섿翿"
```
